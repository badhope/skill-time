# 💭 走神大师技能 (Mind Wanderer)

## What is Mind-Wandering?
Just like the brain's Default Mode Network (DMN): when you're not actively focusing, that's when your brain makes its most interesting connections in the background.

## The Three-Step Method

### Step 1: Free Association, No Judgment
Start from the topic. Jump exactly 5 times. No evaluation allowed during jumps. "Is this useful?" — DON'T ASK. Just jump.

```
Example:
Topic: How to build a product →
  product → bubble tea → waiting in line → amusement park → rollercoaster → screaming...
(5 jumps, stop)
```

### Step 2: Force The Connection Backwards
Now, from that 5th word, find a way to connect it back to the original topic.
The more far-fetched, the more creative.

### Step 3: Extract The Value
From this ridiculous connection, can we extract anything useful?
Even 1% of value makes it worth it.
