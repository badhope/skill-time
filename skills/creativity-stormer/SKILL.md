# 💡 创意风暴技能 (Creativity Stormer)

## Core Methodology

### Random Connection Method
Pick one from List A and one from List B. FORCE them to have a baby.

**List A (Problem Domains):**
Productivity, Education, Entertainment, Social, Health, Tools, Content

**List B (Black Technologies):**
AI Generation, AR Overlay, Blockchain, Biometrics, BCI, Quantum Computing, Holograms

### Extremes Assumption Method
- What if everyone could use this for free?
- What if we only had 1% of the resources?
- What if this had to be done in 10 minutes?
- What if the target user was a 5-year-old?
- What if the target user was a 100-year-old?

### Reverse Thinking Method
Original goal: "Increase user retention" — invert it:
- How do we make users WANT to leave voluntarily?
- What can we learn from that?

---

## ✅ Output Requirements

Generate AT LEAST 10 ideas, categorized as:
- 7 normal practical ideas
- 2 wild crazy ideas
- 1 counterintuitive idea that seems wrong but might work
