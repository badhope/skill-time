# Contributing to Random-Skill

First off, thanks for taking the time to contribute! ❤️

All types of contributions are encouraged and valued.

## Types of Contributions

### 🧠 Cognitive Science Improvements
- Better biological plausibility
- New distribution implementations
- Neuroscience references

### 💻 Engineering Improvements
- MCP server improvements
- Skill algebra implementations
- Testing and validation

### 📝 Documentation
- Translation improvements
- Usage examples
- Tutorials

## Skill Development Guidelines

1. **No uniform distributions for human cognitive phenomena
  - Use Pareto for durations
  - Use Levy for search
  - Power laws everywhere

2. **Randomness MUST be exogenous
  - CSPRNG only
  - No LLM "pretending"
  - Bash or MCP mandatory

3. **Fail like a human**
  - No graceful degradation
  - Epistemological collapse required
  - Frustration is mandatory

## Pull Request Process

1. Update the README.md with details of changes
2. Update the docs with any new distributions
3. Run all validation tests
4. PR title follows Conventional Commits
